===============================================================
J722S Board Configuration Resource Assignment Type Descriptions
===============================================================

Introduction
============

This chapter provides information of Board Configuration resource assignment
type IDs that are permitted in the J722S SoC.  The resource type IDs represent
J722S resources ranges assignable to SoC processing entities (or PEs).


**WARNING**: System Firmware RM currently supports a maximum of 365 RM board
configuration resource assignment ranges on the J722S SoC.  Sending more entries
than the maximum will result in the RM board configuration being NACK'd
