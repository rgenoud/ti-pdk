am6 Board Configuration Resource Assignment Type Descriptions
=============================================================

Introduction
============

This chapter provides information of Board Configuration resource assignment
type IDs that are permitted in the am6 SoC.  The resource type IDs represent am6
resources ranges assignable to SoC processing entities (or PEs).
